4) gestione -> mostra le informazioni dell'utente e un'anteprima dei post creati dall'utente
	URL: BikerBlog/gestione POST   
	INPUT: 
			operazione:"modifica"
			id:3
	OUTPUT:
			redirect alla pagina /Salva nel caso della creazione o modifica di un post oppure 
			stampa "post eliminato" in caso di eliminazione del post

	4.1) Crea post -> mostra i campi contenenti le parti del post da compilare
		URL: BikerBlog/gestione/crea POST  
		INPUP: 
				titolo:"viaggio in Trentino"
				contenuto:"..."
				immagine:trentino.jpg
		OUTPUT: 
			{
				[id:4, titolo:"viaggio in Trentino", descrizione:"...", immagine:trentino.jpg]
			}
			
	4.2) Modifica post -> mostra nel dettaglio il post da modificare
		URL: BikerBlog/gestione/modificapost POST
		INPUT: 
				id:3
		OUTPUT: 
			{
				[titolo:"itinerario in Trentino 3", descrizione:"...", immagine:img.jpg]
			}
	4.3) Salva modifica
		URL: BikerBlog/gestione/salva POST
		INPUT: 
				id:3
				titolo:"itinerario in Trentino 3"
				contenuto:"itinerario di 200Km in Trentino"
				immagine:img.jpg
		OUTPUT:
				"modifica effettuata"
				redirect a /gestione
				
	4.4) Elimina post -> mostra nel dettaglio il post da eliminare
		URL: BikerBlog/gestione/elimina POST
		INPUT: 
				id:2
		OUTPUT: 
				"eliminazione effettuata"
				redirect a /gestione
